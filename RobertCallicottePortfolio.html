Welcome to Battle-Honor!

Embark on an adrenaline-fueled journey with Battle-Honor, a fast-paced fighting game in development for the past 3 months. Designed to connect players globally, it promises real-time, thrilling combat experiences. While the game isn't yet available for download, you can dive into its inner workings through the full source code on my GitHub page, accessible at the top of this site.

This ambitious project springs to life with the help of Unity, a powerful game development platform known for its versatility and user-friendly interface. Unity enables developers to create immersive 2D and 3D experiences, making it an ideal choice for bringing Battle-Honor to life.

The game's backbone is scripted in C#, a language I've recently added to my toolkit. It's not just about the code, though. A significant part of the magic happens through Photon Fusion, a robust netcode framework. Photon Fusion simplifies backend development, letting me concentrate on game creation. However, it's not all smooth sailing; I've dedicated around 85% of my development time to mastering Fusion's API documentation, enhancing my understanding of servers, hosts, clients, and packet management.

This journey has been intense and enlightening, pushing the boundaries of my skills and patience. With continued dedication, I aim to have Battle-Honor ready for playtesting by February 2024. Stay tuned for more updates on this exciting venture!

Lets review some stocks!

Ever been curious about which stock to purchase, but never knew about historical trends? This is the basic premise of my "Stock Tracker". Just type in any ticker you want with the appropriate time frame you're looking for and boom! Instant access to historical trends. All the code for this project can be found on my GitHub link at the top of this webpage.

Well, how does this work, you might ask? This app calls the Yahoo Finance API to retrieve information about the requested ticker a user inputs and then returns back to the view two graphs. One graph displays opening prices, and another shows closing prices within a specified time frame.

The graphs in particular are based on Plotly, which, in my personal opinion, I find more effective than Matplotlib. The combination of Plotly's advanced features and the reliable data from Yahoo Finance API ensures that the information provided is both accurate and visually appealing.

Please feel free to download the project and play around with it yourself! The "Stock Tracker" not only provides valuable insights into stock market trends but also showcases my skills in integrating APIs and data visualization tools effectively.
